# DOMAIN BLOCKER

A lightweight Android application that blocks websites (ALSO ADS TO SOME EXTENT) locally using Android's **VPNService**. The app creates a local VPN to intercept DNS requests and blocks domains from a user-defined list without sending any traffic to external servers. No root access is required, and all filtering is performed entirely on the device.

## Features

* Block websites by domain name
* Fully local VPN-based filtering
* No root required
* No external VPN servers
* Modern UI built with Jetpack Compose
* Persistent storage using Room Database
* Fast and lightweight

## How It Works

1. The app starts a local VPN using Android's `VPNService`.
2. DNS requests generated by the device are intercepted locally.
3. Requested domains are compared against the user's blocked list.
4. Matching domains are blocked, while all other traffic is forwarded normally.

Since all processing occurs on the device, your browsing data never leaves your phone.

## Tech Stack

* Kotlin

* Android VPNService

## Requirements

* Android 10.0 or above
* VPN permission (required for local filtering)

## Privacy

Domain Blocker is designed with privacy in mind.

* No user data is collected.
* No browsing history is stored.
* No internet traffic is sent to third-party servers.
* All domain filtering is performed locally on your device.
